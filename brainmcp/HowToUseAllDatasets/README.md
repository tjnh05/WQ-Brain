# WorldQuant BRAIN 数据集入门指南

本文件夹包含了 WorldQuant BRAIN 平台上所有可用数据集的入门指南，帮助您了解各数据集的基本使用方法。

## 数据集分类

### 1. Analyst 数据集 (分析师数据)
包含分析师预测、评级、ESG评分等信息

### 2. Earnings 数据集 (盈利数据)
包含公司盈利公告、盈利更新邮件等信息

### 3. Fundamental 数据集 (基本面数据)
包含公司财务报表、管理层信息等基本面数据

### 4. News 数据集 (新闻数据)
包含新闻情感、新闻事件等信息

### 5. Other 数据集 (其他数据)
包含其他类型的数据集

## 如何使用

1. 查看对应数据集的 `.md` 文件获取详细信息
2. 每个文件包含：
   - 数据集描述
   - 主要字段说明
   - 使用示例
   - 相关研究论文链接

## 注意事项

- 不同数据集有不同的延迟设置（delay）
- 数据覆盖范围（coverage）可能不同
- 建议在使用前先了解数据集的特点和限制
