# Estimations of Key Fundamentals (analyst14)

## 数据集描述
报告财务报表项目（资产负债表、损益表、现金流量表等）及分析师对即将到来的财政季度/半年度/年度的共识预测值。

## 主要特点
- **全面报表**: 资产负债表、损益表、现金流量表
- **共识预测**: 多家分析师的共识估算
- **全球覆盖**: 国际公司数据
- **历史数据**: 良好的历史记录

## 主要字段
### 财务报表项目
- 资产负债表项目
- 损益表项目
- 现金流量表项目
- 关键财务指标

### 预测数据
- 共识预测值
- 分析师预测分布
- 预测修正历史
- 实际vs预测对比

## 使用场景
1. 基本面分析
2. 预期管理
3. 财务健康评估
4. 跨公司比较

## 使用示例
```
# 预测准确性分析
forecast_accuracy = analyst14_actual - analyst14_consensus_estimate

# 财务健康指标
debt_ratio = analyst14_total_debt / analyst14_total_assets

# 盈利能力
profit_margin = analyst14_net_income / analyst14_revenue
```
