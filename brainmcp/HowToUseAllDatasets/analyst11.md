# ESG Scores (analyst11)

## 数据集描述
提供路透社工具代码(RIC)和彭博标识符(BBID)之间的全面映射，促进跨平台证券识别和数据集成。

## 主要特点
- **标识符映射**: RIC ↔ BBID
- **跨平台支持**: 彭博和Refinitiv
- **高覆盖率和准确性**: 确保数据完整性
- **多源数据整合**: 支持跨供应商数据同步

## 主要字段
- 路透社工具代码(RIC)
- 彭博标识符(BBID)
- 证券映射关系
- 交叉引用信息

## 使用场景
1. 数据集成和同步
2. 多源分析
3. 投资组合管理
4. 回测统一交易池构建
5. 确保数据一致性

## 使用示例
```
# 使用标识符映射进行数据整合
security_mapping = analyst11_map_ric_to_bbid

# 确保数据一致性
consistent_data = analyst11_verify_security_identity
```
