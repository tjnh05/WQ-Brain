# Earnings Forecasts (analyst15)

## 数据集描述
为主要指数和国家提供盈利和其他因素的底部预测数据。在同一组/行业/行业内汇总股票分析，提供组级别的分析师数据。

## 主要特点
- **覆盖范围**: 全球主要指数和国家
- **数据类型**: 组级别汇总数据
- **分析维度**: 组/行业/行业内汇总
- **更新频率**: 定期更新

## 主要字段
- 盈利预测
- 其他财务指标预测
- 组级别汇总数据
- 行业级别汇总数据
- 市场情绪指标

## 使用场景
1. 识别市场估值水平
2. 衡量市场情绪
3. 行业轮动策略
4. 指数成分分析

## 使用示例
```
# 使用组级别盈利预测
group_earnings_forecast = group_rank(analyst15_earnings_forecast, group=industry)

# 结合市场情绪
market_sentiment = ts_mean(analyst15_sentiment_score, 60)
```
