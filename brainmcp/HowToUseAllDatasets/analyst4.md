# Analyst Estimate Data for Equity (analyst4)

## 数据集描述
提供超过16,000家全球公司的日内更新共识财务预测、实际值和公司指导数据，来源于800多家贡献券商。

## 主要特点
- **覆盖范围**: 全球16,000+公司
- **数据源**: 800+贡献券商
- **更新频率**: 日内更新
- **历史数据**: 提供每日历史共识时间线

## 主要字段
- EPS预测和实际值
- 销售额预测
- 股息预测
- 现金流预测
- 行业特定指标

## 使用场景
1. 预测价格变动
2. 识别盈利动量
3. 响应市场驱动事件
4. 分析预测修正和趋势

## 使用示例
```
# 使用EPS预测数据
eps_forecast = rank(analyst4_eps_forecast - analyst4_eps_actual)

# 结合预测修正
eps_revision = ts_rank(analyst4_eps_forecast_change, 20)
```
